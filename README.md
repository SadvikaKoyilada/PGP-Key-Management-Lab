# PGP Key Management Lab

## Overview
This project demonstrates the implementation of OpenPGP encryption using GnuPG (Gpg4win) and Kleopatra.

## Objectives
- Install and configure Gpg4win
- Generate OpenPGP public/private key pairs
- Manage certificates using Kleopatra
- Export public keys securely
- Understand encryption and authentication concepts

## Tools Used
- GnuPG (Gpg4win)
- Kleopatra
- OpenPGP

## Skills Gained
- Public Key Cryptography
- Encryption Fundamentals
- Secure Key Management
- Digital Certificates
- Information Security

## Project Outcome
Successfully generated an OpenPGP key pair, exported a public key certificate, and gained hands-on experience with encryption technologies used for secure communication and authentication.
